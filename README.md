# Gestión de Flores en la Biblioteca

Este proyecto es una aplicación web interactiva para gestionar una biblioteca de flores. Proporciona una interfaz amigable para visualizar, añadir, actualizar, eliminar y buscar flores en la biblioteca, utilizando operaciones CRUD.

## Características

- **Visualización de Flores**: Muestra una lista de flores con detalles como nombre, color, tipo, altura promedio y zona.
- **Añadir Flor**: Permite añadir nuevas flores a la biblioteca mediante un formulario interactivo.
- **Actualizar Flor**: Proporciona una opción para modificar la información de las flores existentes.
- **Eliminar Flor**: Permite eliminar flores de la biblioteca de manera sencilla.
- **Buscar Flores**: Incluye una barra de búsqueda para filtrar flores según diferentes criterios.

## Tecnologías Utilizadas

- **HTML5**: Para la estructura y el contenido semántico de la aplicación web.
- **CSS3**: Para el diseño visual y los estilos de la interfaz de usuario.
- **JavaScript**: Para la lógica del lado del cliente, incluyendo el manejo de eventos y la interacción con la API.
- **Fetch API**: Para realizar solicitudes HTTP a un servidor local (o remoto) que actúa como API REST.
